# vue-webcomponents-sample
Web components sample Todo-list application using vue-cli3 and TypeScript, vue-property-decorator.

## Project setup
```
yarn install
yarn build --target wc-async --name wc-todo 'src/views/Todo.vue'
open sample.html
```
